# 人形机器人中的具身智能智能体：动态时间线知识图谱（最终补充完善版）

这是一个纯 HTML/CSS/JavaScript/SVG 实现的交互式时间线知识图谱网页，主题为：**人形机器人中的具身智能智能体：感知—理解—规划—决策—记忆—世界模型—控制闭环的发展现状与瓶颈**。

本版本在上一版增强图谱的基础上，进一步吸收了用户提供的“深度填充版 data.js”中的高价值内容，并做了三类处理：

1. **合并同义节点**：避免 RT-1/RT-2/OpenVLA/π0/GR00T/Figure/Tesla 等节点重复出现。
2. **新增缺失节点**：补入“大脑—小脑双环架构”“3D 语义场景图”“ACT 动作分块 Transformer”“RMA 快速运动适应”“高低频控制断层”“长程复合误差”等关键节点。
3. **学术化改写**：对过于绝对、夸张或产业界未完全公开验证的说法做中性化处理，并保留“不确定性/需要独立评测”的说明。

## 本地运行

```bash
npm run check
npm run dev
```

然后打开：

## 使用建议

1. 先点击“到底卡在身体还是智能？”查看总论判断。
2. 再点击“大脑—小脑双环分层具身智能体架构”，理解高级认知和底层控制如何分工。
3. 使用“查看前置路径”追溯到控制论、符号 AI、行为式机器人、概率机器人与 TAMP。
4. 使用“查看后续影响”观察 VLA、机器人基础模型、产业平台与瓶颈节点如何展开。
5. 对产业节点请重点阅读“不确定性说明”，不要把公司演示直接等同于规模化自主部署。
